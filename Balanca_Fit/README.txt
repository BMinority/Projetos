# Balança Fit

O Balança Fit é um aplicativo web desenvolvido para calcular o Índice de Massa Corporal (IMC) do usuário. Ele fornece uma avaliação do peso com base na altura e exibe uma mensagem correspondente ao resultado. Além disso, o aplicativo exibe um GIF animado que representa o estado físico do usuário de acordo com o resultado do IMC.

## Funcionalidades

- Calcula o IMC com base no peso e altura inseridos pelo usuário.
- Verifica se os campos de peso e altura estão preenchidos corretamente.
- Fornece mensagens correspondentes ao resultado do IMC, indicando se o usuário está abaixo do peso, com peso adequado, com sobrepeso ou com obesidade de diferentes graus.
- Exibe um GIF animado que representa visualmente o resultado do IMC.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Como usar

1. Faça o download ou clone o repositório do Balança Fit.

2. Abra o arquivo `index.html` em um navegador da web.

3. Preencha o campo "Qual o seu peso?" com o valor correspondente ao seu peso em quilogramas (Kg). Por exemplo, se você pesa 70 kg, insira "70" no campo.

4. Preencha o campo "Qual a sua altura?" com o valor correspondente à sua altura em metros (m). Utilize o formato decimal separando as casas decimais por vírgula. Por exemplo, se você tem 1,70 m de altura, insira "1,70" no campo.

5. Clique no botão "Calcular IMC" para obter o resultado.

6. O resultado do IMC será exibido abaixo do botão, juntamente com uma mensagem correspondente ao estado físico do usuário. Um GIF animado representando o estado físico também será exibido.

7. Para calcular um novo IMC, clique no botão "Reiniciar" para limpar os campos e redefinir as animações.

## Estrutura do Projeto

O projeto Balança Fit é composto pelos seguintes arquivos:

- `index.html`: Contém a estrutura HTML do aplicativo, incluindo os campos de entrada, botões e elementos para exibir o resultado do IMC e as animações.
- `style.css`: Arquivo CSS que define o estilo e o layout do aplicativo.
- `script.js`: Arquivo JavaScript contendo a lógica do cálculo do IMC e o gerenciamento dos eventos do usuário.

## Personalização

Você pode personalizar o aplicativo Balança Fit de acordo com suas preferências. Você pode modificar o estilo e o layout do aplicativo ajustando as regras CSS no arquivo `style.css`. Além disso, você pode adicionar mais animações GIF ou modificar as mensagens correspondentes ao resultado do IMC no arquivo `script.js`.

## Considerações Finais

O Balança Fit é um projeto simples que demonstra como calcular o IMC e fornecer uma avaliação do estado físico do usuário. Sinta-se à vontade para explorar e expandir este projeto de acordo com suas necessidades e conhecimentos. Divirta-se calculando o seu IMC e mantenha-se saudável!
